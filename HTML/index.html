Sobre Mí

Hola, Soy Juan Esteban Ciro Castaño. Soy un desarrollador de software con una sólida formación en análisis y desarrollo de software. Experiencia en Administración y Liderazgo De las experiencias de mi vida, con mayor oportunidad de crecimiento personal, fue como administrador de tienda, donde no solo gestioné el punto de venta, sino que también lideré un equipo de trabajo. Automatización y Desarrollo En mi experiencia en el área de automatización, cree software que optimiza tareas. Trabajé con lenguajes como JavaScript y Node.js, y utilicé herramientas como Selenium WebDriver y XPath para automatizar pruebas de aplicaciones web. Experiencia como QA Tester Durante mis prácticas, tuve la oportunidad de colaborar en la prueba y validación de aplicaciones antes de su lanzamiento, asegurando su calidad y funcionamiento. Usamos herramientas como Azure DevOps para generar reportes y seguimiento de incidencias. Realicé pruebas masivas con JMeter y participé activamente en la creación de manuales de usuario y listas de chequeo. Manejo tecnologías como JavaScript, HTML, CSS, Bootstrap, Angular, Java, Node.js, SQL y MongoDB. Actualmente estoy trabajando en el desarrollo de un sitio web para una empresa dedicada a la venta de productos para mascotas.
